# 🎯 AI 커뮤니티 — 바이브 코딩 시작 가이드

이 파일은 **사용자(당신)** 와 **AI 코딩 에이전트** 모두를 위한 안내문입니다.

---

## 🙋 이게 뭐예요?

**바이브 코딩(Vibe Coding)** 이란 개발 명세를 AI 에이전트에게 통째로 넘기고,
에이전트가 코드를 자동으로 구현하는 방식입니다.

당신은 방금 `AI 커뮤니티_vibe.zip` 하나를 받았습니다. 압축을 풀면 `AI 커뮤니티_vibe/` 폴더 안에
**"우리 프로젝트가 어떤 시스템인지"** 를 체계적으로 정리한 명세서 묶음이 들어 있어요.

이 명세서들을 **AI 코딩 에이전트** (예: Claude Code, Cursor, Codex, Windsurf 등)
에게 통째로 던지면, 에이전트가 **알아서 코드를 작성**해 줍니다.

---

## ✅ 지금 무엇을 해야 하나요? (3단계면 끝!)

### 1단계 — 압축 풀기
받은 `AI 커뮤니티_vibe.zip` 의 압축을 풀면 `AI 커뮤니티_vibe/` 폴더가 생깁니다. (코딩 규칙은 `skills/` 폴더에 이미 들어 있습니다 — AI 에이전트가 자동으로 참조합니다.)

### 2단계 — Claude Code 로 폴더 열기
터미널에서 `cd "AI 커뮤니티_vibe"` 로 폴더에 들어간 뒤 `claude` 를 실행하세요. Claude Code 가 **CLAUDE.md 를 자동으로 읽습니다.**

> 아직 도구가 없다면: **Claude Code**(`https://claude.com/code`, 터미널) 또는 **Cursor**(`https://cursor.com`, 편집기)를 설치하세요.

### 3단계 — "시작해줘" 한 마디
폴더를 열면 에이전트가 `CLAUDE.md` 를 **자동으로 읽어** 무엇을 할지 이미 압니다. 채팅에 이렇게만 입력하세요:
```
시작해줘
```
> 혹시 자동으로 안 읽는 도구라면 아래 한 줄을 붙여넣으세요:
> `이 폴더의 CLAUDE.md 와 0_START_HERE.md 를 읽고 Phase 0(규칙 내재화) 부터 진행해줘. 각 단계 끝나면 확인받아줘.`
`skills/` 폴더의 모든 .md 파일을 코딩 규칙 참조집으로 활용해줘 — 코드를 작성하기 전 관련 규칙을 먼저 확인하고 적용해줘.

이후 에이전트가 **Phase 별로 코드를 만들고 보고**합니다. 각 단계에서 결과를 확인하고, 마음에 안 들면 자연어로 수정 요청하세요.
완전히 자동으로 한 번에 만들어주는 게 아니라 **단계별 협업**입니다.

🏁 **모두 끝나면**: GitHub 업로드는 **에이전트가 `SHIP_TO_HARNESS.md` 절차대로 대신 처리**합니다 — 계정이 없어도 가입 화면에서 몇 번 클릭만 하면 돼요. 업로드가 끝나면 Harness의 **결과물(Deliverables) 화면**에 그 저장소를 등록한 뒤 **Lint 화면**에서 검증을 실행해 점수를 받으세요. 점수가 낮으면 수정 지시서까지 만들어 드립니다. 여기까지가 진짜 완주입니다.
